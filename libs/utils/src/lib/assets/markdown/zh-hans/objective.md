# 目标

这个项目的目标，是让用户能够创建并管理自己的 Google Tag Manager（GTM）标签审计项目。用户可以创建项目、加入标签，并对标签执行审计。审计会检查标签是否在正确场景下被使用，以及是否确实有被触发，之后再根据结果调整标签配置。

## Google Tag Manager

Google Tag Manager 是一套标签管理系统，让用户可以在不修改网站源代码的前提下，管理与部署网站上的标签。这些标签通常用于收集数据，并发送到 Google Analytics、Facebook Pixel 等第三方服务。

## 审计

审计会确认标签是否在正确场景下被使用，以及是否真的有被触发。例如，如果某个标签应该只在特定页面触发，系统就会检查它是否确实在该页面启动。如果标签没有按预期执行，报告会标记为失败并附上原因。

可能的失败原因包括：

- 标签完全没有触发
- 标签虽然触发，但参数错误或放错位置

TagCheck 让你可以通过自动化方式完成这类项目审计。
