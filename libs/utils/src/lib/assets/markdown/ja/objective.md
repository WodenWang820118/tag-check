# 目的

このプロジェクトの目的は、ユーザーが自分自身の Google Tag Manager（GTM）タグ監査プロジェクトを作成し、管理できるようにすることです。ユーザーはプロジェクトを作成し、タグを追加し、そのタグに対して監査を実行できます。監査では、タグが正しい文脈で使用されているか、あるいはまったく使用されていないかを確認し、その結果をもとに必要な修正を行えます。

## Google Tag Manager

Google Tag Manager は、Web サイトのコードを直接変更することなく、タグを管理および配信できるタグ管理システムです。タグはデータ収集のためのコード断片であり、Google Analytics や Facebook Pixel などの外部サービスにデータを送るために利用されます。

## 監査

監査では、タグが正しい文脈で使われているか、また実際に発火しているかを確認します。たとえば、あるタグが特定のページでのみ発火すべき場合、そのページで本当に発火しているかをチェックします。期待どおりに発火していない場合、レポートでは失敗として記録され、理由も提示されます。

失敗理由の例:

- タグがまったく発火していない
- タグは発火しているが、パラメータが誤っている、または位置が不適切

TagCheck を使うことで、こうした監査を自動化して実施できます。
