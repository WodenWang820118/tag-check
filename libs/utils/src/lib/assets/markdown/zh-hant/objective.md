# 目標

這個專案的目標，是讓使用者能建立並管理自己的 Google Tag Manager（GTM）標記稽核專案。使用者可以建立專案、加入標記，並對標記執行稽核。稽核會檢查標記是否在正確情境下被使用，以及是否確實有被觸發，之後再依據結果調整標記設定。

## Google Tag Manager

Google Tag Manager 是一套標記管理系統，讓使用者可以在不修改網站原始碼的前提下，管理與部署網站上的標記。這些標記通常用來收集資料，並傳送到 Google Analytics、Facebook Pixel 等第三方服務。

## 稽核

稽核會確認標記是否在正確情境下被使用，以及是否真的有被觸發。舉例來說，如果某個標記應該只在特定頁面觸發，系統就會檢查它是否確實在該頁面啟動。若標記沒有如預期執行，報告會標記為失敗並附上原因。

可能的失敗原因包括：

- 標記完全沒有觸發
- 標記雖然觸發，但參數錯誤或放錯位置

TagCheck 讓你可以透過自動化方式完成這類專案稽核。
