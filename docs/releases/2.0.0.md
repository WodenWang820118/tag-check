# Tag Check 2.0.0

Tag Check `2.0.0` is the first official desktop release built on the Tauri
runtime. This release keeps the existing verification product surface while
changing the desktop packaging model from the previous desktop approach to a
cross-platform Tauri release contract.

## Highlights

- Introduces the first official Tauri desktop release for Tag Check.
- Publishes a coordinated desktop bundle set for Windows, macOS, and Linux.
- Standardizes release artifact naming, checksum generation, and draft GitHub
  Release publication.

## Desktop Assets

- Windows x64 NSIS installer
- macOS arm64 unsigned `.app` archive packaged as `.tar.gz`
- Linux x64 AppImage

## Operator Notes

- The macOS asset is intentionally unsigned in `2.0.0`. Testers may need to
  use manual trust or Gatekeeper override steps before opening it.
- The GitHub Release is created as a draft so operators can review the attached
  assets and notes before publishing.
- `SHA256SUMS.txt` is attached alongside the release assets and covers every
  packaged desktop artifact in this release.
